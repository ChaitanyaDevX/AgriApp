# 🌾 AgriApp

An Android application built to empower farmers and agricultural enthusiasts with smart tools, crop information, and farm management features — all from the palm of their hand.

---

## 📱 About the Project

AgriApp is a Java-based Android application developed in Android Studio. It bridges the gap between modern technology and agriculture, providing farmers with easy access to tools they need to make informed decisions and manage their work efficiently.

---

## ✨ Features

- 🌱 **Crop Information** – Browse detailed information on various crops including growth tips and care guides
- 🌦️ **Weather Updates** – Get current weather conditions relevant to farming decisions
- 🛒 **Marketplace** – Explore and purchase agricultural products, seeds, and tools
- 📰 **Agri News** – Stay updated with the latest news and developments in the agriculture sector
- 👥 **Farmer Community** – Connect and interact with other farmers
- 📋 **Task Manager** – Manage daily farming tasks and reminders

## 🚀 Getting Started

### Prerequisites

- Android Studio (latest stable version recommended)
- Android SDK (API level 21+)
- Java 8 or higher

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/ChaitanyaDevX/AgriApp.git
   ```

2. **Open in Android Studio**
   - Launch Android Studio
   - Click **File → Open** and select the cloned `AgriApp` folder

3. **Sync Gradle**
   - Click **Sync Now** when prompted, or go to **File → Sync Project with Gradle Files**

4. **Run the App**
   - Connect an Android device or start an emulator
   - Click the **Run ▶** button or press `Shift + F10`
